## üéØ Screenshot Ideas (Pick 3-5)

### Screenshot 1: Word Tooltip (Essential)
**What to capture:**
- A webpage with text (Wikipedia article, news site, blog)
- A word selected/highlighted
- The Nimbus tooltip showing:
  - Word definition
  - Pronunciation
  - Synonyms (scrollable)
  - Copy, Favorite, Search buttons

**How to take:**
1. Open a Wikipedia article (e.g., https://en.wikipedia.org/wiki/Technology)
2. Select a word (1-2 words max)
3. Wait for tooltip to appear
4. Take screenshot of entire browser window
5. Crop to show: webpage + tooltip prominently

---

### Screenshot 2: Hub Dashboard (Essential)
**What to capture:**
- Click extension icon to open hub
- Show:
  - Word of the Day card
  - Favorites section (if you have some)
  - Recent Searches section
  - Search bar at top

**How to take:**
1. Click Nimbus extension icon
2. Hub popup opens
3. Take screenshot of the popup
4. May need to resize browser window to show popup clearly

**What to capture:**

## üõ†Ô∏è How to Take Screenshots

### Option 1: macOS Built-in Screenshot Tool
1. **Full screen:** `Shift + Command + 3`
2. **Selected area:** `Shift + Command + 4` (then drag to select)
3. **Window:** `Shift + Command + 4` then press `Space` and click window

### Option 2: Chrome DevTools
1. Open DevTools (F12)
2. Click device toolbar icon (or `Cmd + Shift + M`)
3. Set custom size: 1280x800
4. Take screenshot: `Cmd + Shift + P` ‚Üí type "screenshot" ‚Üí select "Capture screenshot"

### Option 3: Browser Extension
- Use a screenshot extension like "Awesome Screenshot" or "Nimbus Screenshot"

### Required Edits:
