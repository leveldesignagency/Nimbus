## ğŸ› ï¸ How to Create

### Option 1: Canva (Easiest)
1. Go to canva.com
2. Create custom size: 1280x800
3. Add your logo
4. Add text: "Nimbus" + tagline
5. Use navy blue background
6. Download as PNG

### Option 2: Figma (Free, Professional)
1. Go to figma.com
2. Create new design file
3. Set frame to 1280x800
4. Add logo and text
5. Export as PNG

### Option 3: Photoshop/Photopea
1. Create new document: 1280x800
2. Add navy blue gradient background
3. Place logo
4. Add text layers
5. Export as PNG

## ğŸ¨ Design Specifications

**Background:**

- Size: ~200-300px width (adjust to fit)

- "Nimbus" - Large, bold (60-80px)
- Tagline - Medium (24-32px)

- Center everything vertically and horizontally
- Leave padding around edges (50-100px)
